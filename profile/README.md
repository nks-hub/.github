## NKS Hub

Open-source balíčky a rozšíření pro **Nette Framework**, TypeScript SDK a nástroje.

### Nette Extensions

| Balíček | Popis |
|---------|-------|
| [nette-cloudflare-r2](https://github.com/nks-hub/nette-cloudflare-r2) | Cloudflare R2 storage — S3-compatible upload/download, multipart, presigned URLs, Tracy panel |
| [nette-bankid](https://github.com/nks-hub/nette-bankid) | Czech BankID OAuth2/OIDC autentizace — ověření identity přes internetové bankovnictví |
| [nette-sso](https://github.com/nks-hub/nette-sso) | SSO přes OAuth2/OpenID Connect (Authentik) — role mapping, CSRF, admin groups |
| [nette-ruian](https://github.com/nks-hub/nette-ruian) | RUIAN API — validace adres, geokódování, progressive address builder |

### SDK & Tools

| Balíček | Popis |
|---------|-------|
| [rybbit](https://github.com/nks-hub/rybbit) | Privacy-first analytics SDK pro TypeScript — GA4 events, GTM bridge, zero cookies |

### Instalace

```bash
# Nette extensions (Composer)
composer require nks-hub/nette-cloudflare-r2
composer require nks-hub/nette-bankid
composer require nks-hub/nette-sso
composer require nks-hub/nette-ruian

# TypeScript SDK (npm)
npm install @nks-hub/rybbit
```
